Flutter project.

## Getting Started
This is a mobile application designed to streamline the clothing selection process and boost users confidence in their style. StyleShuffle combines neural networks, two APIs, and high-performance NoSQL.

API integration - this stage is dedicated to integrating external services and data through APIs. I have developed flexible and efficient interfaces to facilitate communication between the StyleShuffle application and external data sources. This allows the StyleShuffle app to obtain up-to-date information about weather, fashion trends, and other important parameters.

Neural Networks - at this stage, convolutional neural networks have been created and trained to classify various types of clothing into categories, taking into account weather conditions.

NoSQL - at this stage, the selection and implementation of a non-relational database take place, ensuring high performance and scalability of the application.
